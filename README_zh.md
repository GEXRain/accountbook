#### 介绍

本项目是一款简易记账软件，实现了一些日常记账需求，并采用在线数据库的方式存储数据，项目的主要意义在于练习实践所学Java编程方法。

#### 软件架构
本项目由Java实现，UI使用Javafx驱动，使用Maven管理项目。


#### 安装教程

使用Maven构建项目即可。

1.  登录账户
2.  创建资产账户
3.  添加账单
